## HUGO 安裝
#### macOS
利用 [Homebrew](https://brew.sh/) 安裝  
```bash
brew install hugo
```

![](/images/hugo/02-install/install-01-macOS.png)


### Ubuntu
利用 [apt](https://ubuntu.com/server/docs/package-management) 安裝
```bash
sudo apt install hugo -y
```

![](/images/hugo/02-install/install-02-ubuntu.png)
